# 32K 全能模式配置文档

## 🎬 模式概述

**32K 全能模式** 是专为 **AI 漫剧/短剧/复杂场景** 设计的黄金配置，平衡了性能、容量和成本，覆盖 95%+ 的视频生成场景。

---

## 📊 核心参数

| 参数 | 配置值 | 说明 |
|------|--------|----|
| **上下文窗口** | 32K tokens | 黄金尺寸，适合复杂场景 |
| **缓存大小** | Large | 大缓存，平衡容量和速度 |
| **Batch Size** | 1536 | 较大 batch，并行能力强 |
| **量化方式** | Q4_K_M | 4 位整数，精度损失<1% |
| **训练步数** | 18,000 步 | 充分优化 |
| **梯度累积** | 1 | 无需累积，GPU 够用 |
| **学习率** | 1.8e-5 | 稍高，加快收敛 |
| **自动训练** | true | 每 50 次生成自动训练 |

---

## 🎯 适用场景

### **✅ 强烈推荐**

1. **AI 漫剧/短剧**
   - 多场景切换 (5-10 个场景)
   - 角色一致性 (完整描述角色、服装、表情)
   - 复杂光线和氛围

2. **电商产品展示**
   - 精细控制场景、灯光、角度
   - 多产品展示
   - 产品细节特写

3. **复杂场景描述**
   - 角色、动作、环境综合描述
   - 需要完整上下文支持
   - 30 秒 -3 分钟视频

4. **中短视频制作**
   - 抖音/TikTok 中长视频
   - 教育/科普视频
   - 品牌宣传视频

### **⚠️ 不适用场景**

- ❌ 6-15 秒短视频 (用 16K 极致模式)
- ❌ 简单产品展示 (用 16K 极致模式)
- ❌ 需要极致速度 (用 16K 极致模式)

## 📝 配置详情

### **1. 配置文件位置**

```
src/ai-video-generation/mode-32k.js
```

### **2. 核心配置对象**

```javascript
// src/ai-video-generation/mode-32k.js
const arealConfig = {
  context: '32k',              // 32K 上下文
  buffer: 'large',             // 大缓存
  batch: 1536,                 // 较大 batch
  quantization: 'q4_k_m',      // 4 位量化
  training_steps: 18000,       // 18000 步
  gradient_accumulation: 1,    // 无需累积
  learning_rate: 1.8e-5,       // 稍高学习率
  auto_train: true,            // 自动训练
  train_interval: 50           // 每 50 次生成
};
```

### **3. 推荐参数说明**

#### **context: '32k'**
- ✅ **覆盖 95%+ 视频场景**
- ✅ **角色一致性**: 可容纳完整角色描述
- ✅ **多场景切换**: 5-10 个场景无缝衔接
- ✅ **精细控制**: 光线、颜色、动作等细节

#### **buffer: 'large'**
- ✅ **容量**: 比 16K 多 4-6GB
- ✅ **性能**: 速度提升 4.0-4.5x
- ✅ **性价比**: 72-75% 成本节省

#### **batch: 1536**
- ✅ **并行能力**: 中等偏上
- ✅ **GPU 利用率**: 95-97%
- ✅ **稳定性**: 无需梯度累积

#### **quantization: 'q4_k_m'**
- ✅ **精度损失**: <1%
- ✅ **兼容性**: 完全支持 AReaL
- ✅ **速度**: 4 位整数加速

## 🔧 高级配置

### **1. 自定义 batch 大小**

```javascript
// 需要更高性能时
const config = {
  ...defaultConfig,
  batch: 2048  // 增加 batch，GPU 利用率更高
};
```

**注意**: batch > 2048 可能增加显存占用至 17-18GB

### **2. 调整学习率**

```javascript
// 需要更快收敛时
const config = {
  ...defaultConfig,
  learning_rate: 2.0e-5  // 稍高学习率
};
```

**注意**: 学习率过高可能导致训练不稳定

### **3. 自定义训练步数**

```javascript
// 需要更多优化时
const config = {
  ...defaultConfig,
  training_steps: 25000  // 增加训练步数
};
```

**注意**: 更多训练步数 = 更长训练时间，但效果更佳
